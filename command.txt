To Create the project 
    python3 django-admin startproject portfolio .
        to have all the project file's inside the folder put "." 

To run the server 
    python3 manage.py runserver


To create a django App
    python3 manage.py startapp appname
        appname should be plural
        dont use keyword of python3

TO make Migrate (it is nothing but the changes done in model file it will refelect the changes to the DB) 
    python manage.py migrate
    python manage.py makemigrate    # doesnt apply changes in the migration.
    python manage.py makemigrations {appname}   # doesnt apply changes in the migration.

To Create Superuser
    python manage.py createsuperuser
